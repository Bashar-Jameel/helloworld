# Website gemaakt

Ik heb dit project ontwikkeld in opdracht van een klant. Het betreft een extern project dat speciaal voor deze klant is gemaakt.
Bij de ontwikkeling heb ik gebruikgemaakt van verschillende technieken, waaronder PHP, HTML, Bootstrap en JavaScript.

Het doel van dit project was om een functionele en gebruiksvriendelijke oplossing te realiseren die aansluit bij de wensen van de klant.
